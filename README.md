# How to use project

kickstart nextjs 7.0.2

## Installation

Step 1 - Clone repo

```bash
git clone https://github.com/benzintel/nextjs-react
```

Step 2 - Install packet

```bash
npm install
```

Step 3 - Copy .env.example to .env

```bash
cp .env.example .env
```

Step 4 - Run service 

```bash
npm run dev
```
